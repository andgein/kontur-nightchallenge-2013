Память. Циклически замкнута. В памяти команды процессору. Одна программа ваша, одна вражеская. В конце должна остаться только одна!

Спецификация

Must read Core War для чайников

Must have Исходники и среда для запуска

↑ это полная копия того, что работает на этом сервере. Его можно использовать для локальной отладки, если захотите. Отправлять бота тем не менее нужно на https://corewar.skbkontur.ru/submit.html :-)

Вы можете запустить файл bin/corewar.exe, который поднимет локальный HTTP-сервер на порту 8080.
